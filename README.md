# Production-Style CI/CD Platform (AWS Free Tier)

## Overview
Built a production-style deployment pipeline with automated health checks and rollback capability.

## Key Features
- Terraform-managed infrastructure
- Versioned deployments with symlink switching
- Automated CI/CD using GitHub Actions
- Health checks and rollback on failure
- Monitoring with CloudWatch

## Why this matters
This project focuses on reliability and safe deployments, reflecting real-world DevOps practices.
